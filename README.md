这个系列会基于 JavaScript 描述数据结构与算法。

### 思考

就我现在的思考来说，学习数据结构与算法可以基于任何一个成熟的语言上（挑自己熟悉的语言来做）。

数据结构在哪里能够应用到要注意总结。

用 JavaScript 学完一遍后可以用别的语言再来一遍，比如 C。
